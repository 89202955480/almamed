Описание продукта:

  Скрипт предназначен для создания резервных копий веб сайтов и баз данных на удаленные сервера такие как Яндекс.диск.

Принцип работы:

  После установки и настройки плагина, владелец сайта сможет создавать резервные копии сайта и баз данных автоматически.

Установка:

  1. Скопируйте папку модуля /autobackupmaster в корень вашего сайта
  2. Дайте права 666 файлу /config.php
  3. Дайте права 777 папкам /backup_tmp и /abm_backup
  4. Запустите инсталяцию http://ваш сайт.ru/autobackupmaster/install/.
     !!!Внимание!!! При инсталяции указывайте логин и пароль пользователя root к базе данных!
  5. После инсталяции удалите папку /install и дайте файлу /config.php права 644
  6. Залогинтесь http://ваш домаин.ру/autobackupmaster/
  7. Установите и настройте требуемые модули http://ваш домаин.ру/autobackupmaster/?tab=modules
  8.a
     Для бекапа базы данных, установите соответствующий модуль и укажите какие базы данных бекапить.
  8.b
     Для бекапа файлов, установите соответствующий модуль и укажите какие папки бекапить,
     а какие исключать из бекапа.
  9.a
     Для бекапа на Я.Диск 2OAuth, установите соответствующий модуль, получите и введите ключ авторизации модуля.
  9.b
     Для бекапа по FTP, установите соответствующий модуль, укажите хост, пароль и логин для FTP.
     Создайте папку для бекапа на удаленном сервере, и укажите в настройках модуля путь к ней.
  9.c
     Для локального бекапа, установите соответствующий модуль, укажите права 777 для папки abm_backup.
     Укажите в настройках модуля путь к ней. Если хотите получать бекап базы данных на емаил, то укажите это.
  9.d
     Для бекапа на Dropbox, установите соответствующий модуль, получите и введите ключ авторизации модуля.
  9.e
     Для бекапа на Я.Диск WebDav, установите соответствующий модуль, укажите пароль и логин вашего Я.Диск.
     Создайте на Я.Диск папку для бекапа, и укажите в настройках модуля путь к ней.
  10. Создайте Cron задачу для файла /backup_cron.php рекомендуемый интервал 24 часа

    Установка завершена!
